👋 Sobre mim
Sou um Desenvolvedor Full Stack Júnior apaixonado por construir aplicações web e mobile que resolvem problemas reais. Formado em Análise e Desenvolvimento de Sistemas pelo IFFAR, tenho experiência prática com React, Next.js, Node.js e Flutter.
Meu projeto mais marcante foi o ContoTEA — um aplicativo mobile com IA para engajar crianças com Transtorno do Espectro Autista, premiado com 🥈 2º lugar no MEPT (Mostra de Educação, Pesquisa e Tecnologia).
